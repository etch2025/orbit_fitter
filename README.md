Orbit Fitter for binary stars.

General Notes:
- Primary star is located at origin
- Uses NumPy, Matplotlib, and SciPy libraries to generate an elliptical orbit
- Plots obervation data, fitted orbit, and apoapsis and periapsis location
- Example binary star data from [Stellie Doppie](https://www.stelledoppie.it/) included
    - 61 Cygni AB
    - Kruger 60 (DO Cephei)
    - Sirius
    - Porrima (Gamma Virginis)

Planned Features
- Predicted Position
- Parabolic/Hyperbolic orbit fitting
- Past and predicted time of apoapsis and periapsis
- Roche Lobe location (for interferometic data)
